Documentation pour le Mirage fest

Créer une typo

La seule chose a faire pour créer un typographie paramètrer est d'appeler la fonction Ptypo.createFont( nomTypo, template );.

template peut prendre les valeurs suivantes: 'venus', 'fell', 'elzevir'

'elzevir' est encore en développement et donc incomplet.


Une fois la fonction appelée, une nouvelle font-family est créée portant le nom nomTypo. Pour utiliser cette nouvelle font-family il suffit dans un fichier css de créer une classe ou un id utilisant cette font-family.

Example

index.html


<!doctype html>
<html>
	<head>
		<link rel="stylesheet" href="fest.css"></link>
	</head>
	<body>
		<div id="typo">
			Hello world!
		</div>
		<script src="prototypo.js"></script>
		<script src="prototypo-canvas.js"></script>
		<script src="src/lib.js"></script>
		<script src="mirage.js"></script>
	</body>
</html>

mirage.js


Ptypo.createFont('typo');

fest.css


#typo {
	font-family: typo;
}

Result

Hello world!

Changer une typo

Pour changer une typo il suffit d'appeler la fonction Ptypo.changeParam( value, paramName, nomTypo);

Pour paramétrer par défaut les paramètres d'une typo on peut ajouter: 

					
Ptypo.createFont(nomTypo, template).then( function() {
	Ptypo.changeParam( value, paramName, nomTypo);
});

paramName peut prendre les valeurs suivantes

Nom du paramètre Changement sur la typo
thickness Change la graisse de la typo
width Change la largeur des caractères
xHeight Change la hauteur des caractères
slant Change l'inclinaison des caractères
etc. Une vingtaine de paramètres sont accessibles. Parlons-en ensemble!

Example

index.html


<!doctype html>
<html>
	<head>
		<link rel="stylesheet" href="fest.css"></link>
	</head>
	<body>
		<div id="typo2">
			Hello world!
		</div>
		<div id="plus">+</div>
		<div id="minus">-</div>
		<script src="prototypo.js"></script>
		<script src="prototypo-canvas.js"></script>
		<script src="src/lib.js"></script>
		<script src="mirage.js"></script>
	</body>
</html>

mirage.js


Ptypo.createFont('typo2');

document.getElementById('plus').addEventListener('click', function() {
	var newParam = Ptypo.getParam('thickness', 'typo2') + 10;
	Ptypo.changeParam(newParam, 'thickness', 'typo2');
});

document.getElementById('minus').addEventListener('click', function() {
	var newParam = Ptypo.getParam('thickness', 'typo2') - 10;
	Ptypo.changeParam(newParam, 'thickness', 'typo2');
});

fest.css


#typo {
	font-family: typo2;
}

Result

Hello world!
